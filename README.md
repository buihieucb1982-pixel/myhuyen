# 🌙✨ Nguyễn Thị Khánh Huyền - Personal Portfolio Website

## 🎯 Overview

A beautiful, modern, and fully responsive personal portfolio website for a Creative UI/UX Designer and Digital Graphics student. The website features a dreamy aesthetic with pink & white colors, smooth animations, and interactive productivity tools.

## 🎨 Design Features

### Visual Design
- **Color Scheme**: Soft Pink, White, Pastel Purple, Moonlight Blue
- **Theme Support**: Light/Dark mode toggle
- **Aesthetic**: Dreamy, artistic, modern, elegant, and soft UI/UX
- **Animations**: Multiple CSS animations including glow, pulse, float, sparkle, twinkle, bounce, and more

### Key Sections
1. **Navigation Bar** - Fixed navbar with logo, menu links, and theme toggle
2. **Hero Section** - Introduction with animated decorations and call-to-action buttons
3. **About Section** - Personal story and career goals
4. **Skills Section** - Animated progress bars showing proficiency levels
5. **Projects Section** - Showcase of 3 featured projects
6. **Experience Section** - Published works and achievements
7. **Productivity Dashboard** - Interactive tools (clock, timer, deadlines, notes)
8. **Contact Section** - Contact information
9. **Footer** - Credits

## 🚀 Technologies Used

- **HTML5** - Semantic markup
- **CSS3** - Advanced styling with animations and glassmorphism
- **JavaScript (Vanilla)** - No dependencies required

## 📱 Responsive Design

- Desktop, Tablet, and Mobile optimized
- Mobile-first approach
- Touch-friendly interface

## ✨ Features

### Theme Toggle
- Light/Dark mode with smooth transitions
- Preference saved to LocalStorage

### Productivity Dashboard
1. **Real-Time Clock** - Digital & analog display
2. **Pomodoro Timer** - 25-min focus sessions with notifications
3. **Deadline Manager** - Track tasks with countdown
4. **Sticky Notes** - Create and save notes locally

### Animations
- 🌙 Floating moon, ⭐ twinkling stars, 🧸 bouncing bear
- Smooth page transitions and hover effects
- Glow effects and particle animations

## 📖 Usage Tips

1. **Theme Toggle**: Click moon icon in navbar (Alt + T)
2. **Timer**: Use preset buttons (25m, 5m, 10m)
3. **Notes**: Auto-saved to browser storage
4. **Deadlines**: Automatic countdown tracking
5. **Navigation**: Smooth scroll to sections
