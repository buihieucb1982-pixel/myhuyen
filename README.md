🎨 Main Design Prompt

You are a Senior UI/UX Designer and Front-end Developer.

Create a modern personal portfolio website using 
HTML, CSS and JavaScript.

Style:
- Minimal Modern
- Pink + White aesthetic
- Feminine and elegant UI
- Soft shadows and gradients
- Responsive design
- Smooth animation
- Creative designer portfolio feeling

🌸 Hero Section Prompt

Design a modern hero section for a portfolio website 
with profile image, introduction text, CTA buttons 
and floating background shapes.

Style:
- Elegant spacing
- Soft pink gradient
- Modern typography
- Creative UI/UX feeling

💻 Responsive Design Prompt

Optimize the portfolio website for responsive design 
across desktop, tablet and mobile devices 
using Flexbox and Grid layout.

✨ Animation Prompt

Add smooth scrolling animation, hover effects, 
floating shapes and fade-in transitions 
for a modern portfolio experience.

🌙 Dark Mode Prompt

Create a dark mode and light mode toggle 
with smooth transition effects 
using JavaScript and CSS variables.

📝 Productivity Widget Prompt

Create productivity widgets including:
- Realtime clock
- Deadline countdown timer
- Sticky notes widget

Style:
- Pastel cards
- Glassmorphism UI
- Modern minimal design

🛠 AI Refinement Prompt

Improve spacing, typography, visual hierarchy 
and responsive experience to achieve 
a modern creative designer portfolio style.

🌟 Final Design Goal

Create a modern, elegant and creative digital portfolio 
that reflects personal identity, UI/UX thinking 
and frontend development skills.
